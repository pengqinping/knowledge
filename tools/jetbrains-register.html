Jetbrains 系列破解

获取 jetbrains-agent.jar

  • 百度获取，放到对应 App 的 bin 目录，点击下载
  • Appliction -> 显示包内容 -> Contents -> bin
  • License server address： http://fls.jetbrains-agent.com

修改 Custome VM Option

  • Help -> Custome VM Option
  • 在最下面下面一段代码，注意路径是 我们上面一部放入的文件

PyCharm

-javaagent:/Applications/PyCharm.app/Contents/bin/jetbrains-agent.jar

IntelliJ IDEA

-javaagent:/Applications/IntelliJ IDEA.app/Contents/bin/jetbrains-agent.jar

PhpStorm

-javaagent:/Applications/PhpStorm.app/Contents/bin/jetbrains-agent.jar

RubyMine

-javaagent:/Applications/RubyMine.app/Contents/bin/jetbrains-agent.jar

CLion

-javaagent:/Applications/CLion.app/Contents/bin/jetbrains-agent.jar
